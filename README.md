#  Lekal Engine V4.0
**The Logic Behind Realify Games.**

Lekal is a custom-built, high-performance scripting language designed for rapid game development. It simplifies complex logic into human-readable commands.

##  Core Functions
- **Chained Logic:** Use `&` to run multiple commands at once.
- **Event Triggers:** Use `when touched [object]:` to handle game collisions.
- **Dynamic Memory:** Integrated system for `health`, `score`, and `player_name`.
- **User Input:** Use `variable = input >> "Message"` to interact with players.

##  Installation
1. Download `lekal-eng`.
3. Run the script using **Python 3.10+**.
4. Start coding in the `Lekal >>` prompt.

## 📜 Example Script
`name = input >> "Enter Name:" & say >> "Welcome" & say >> name`
